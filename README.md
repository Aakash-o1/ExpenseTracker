# 💸 Expense Tracker Web App

A modern, responsive expense tracking web application built with **React**, **TypeScript**, and **Vite**, styled using **Tailwind CSS**. This app helps users efficiently manage their daily expenses, track spending habits, and stay financially organized.

---

## 🚀 Tech Stack

* **Frontend:** React + TypeScript
* **Styling:** Tailwind CSS
* **Build Tool:** Vite
* **Markup & Structure:** HTML5 + CSS3

---

## ✨ Features

* ➕ Add, edit, and delete expenses
* 📊 Track total spending in real-time
* 🗂️ Categorize expenses (e.g., Food, Travel, Bills)
* 🔍 Filter and search expenses
* 📱 Fully responsive design
* ⚡ Fast performance with Vite
* 🎯 Type-safe development with TypeScript

## ⚙️ Installation & Setup

1. **Clone the repository**

   ```bash
   git clone https://github.com/your-username/expense-tracker.git
   cd expense-tracker
   ```

2. **Install dependencies**

   ```bash
   npm install
   ```

3. **Run development server**

   ```bash
   npm run dev
   ```

4. Open in browser:

   ```
   http://localhost:5173
   ```

---

## 🛠️ Build for Production

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```
